## 🚀 立即使用

### 當前狀態

✅ 前端服務正在背景運行
✅ 訪問地址: **http://localhost:3000**

### 如何測試

1. **開啟瀏覽器** 訪問 http://localhost:3000

2. **選擇範例**
   - 點擊「選擇範例」下拉選單
   - 選擇任一範例 (例如: 「產品標籤」或「資產標籤」)
   - 編輯器會自動填入 TSPL 代碼

3. **預覽標籤**
   - 點擊「預覽」按鈕
   - 右側會顯示標籤渲染效果

4. **自訂標籤**
   - 直接在編輯器中修改 TSPL 代碼
   - 點擊「預覽」查看結果

## 💡 快速修改指南

### 修改文字

```tspl
TEXT 100,100,"3",0,1,1,"您的文字"
     │   │   │  │ │ │  └─ 文字內容
     │   │   │  │ │ └─ Y 軸放大
     │   │   │  │ └─ X 軸放大
     │   │   │  └─ 旋轉角度 (0, 90, 180, 270)
     │   │   └─ 字型編號
     │   └─ Y 座標
     └─ X 座標
```

### 修改條碼

```tspl
BARCODE 100,140,"128",50,0,0,1,2,"您的條碼"
        │   │   │    │  │ │ │ │  └─ 條碼內容
        │   │   │    │  │ │ │ └─ 寬條寬度
        │   │   │    │  │ │ └─ 窄條寬度
        │   │   │    │  │ └─ 旋轉
        │   │   │    │  └─ 是否顯示文字
        │   │   │    └─ 高度
        │   │   └─ 條碼類型 (128, 39, EAN13...)
        │   └─ Y 座標
        └─ X 座標
```

### 修改 QR Code

```tspl
QRCODE 150,100,H,5,A,0,"您的網址"
       │   │   │ │ │ │  └─ QR Code 內容
       │   │   │ │ │ └─ 旋轉
       │   │   │ │ └─ 模式 (A=自動, M=手動)
       │   │   │ └─ 大小
       │   │   └─ 糾錯等級 (L, M, Q, H)
       │   └─ Y 座標
       └─ X 座標
```

## 🔧 進階使用

### 停止前端服務

在運行前端的終端視窗中按 `Ctrl + C`

### 重新啟動

```bash
cd frontend
npm start
```

### 建置生產版本

```bash
cd frontend
npm run build
```

建置後的檔案在 `frontend/build/` 目錄,可部署到任何靜態網站。

## ❓ 常見問題

### Q: 為什麼有些 TSPL 指令沒效果?

A: 目前支援基本指令 (SIZE, GAP, TEXT, BARCODE, QRCODE, BOX, BAR)。部分進階指令可能尚未實作。

### Q: 可以匯出標籤嗎?

A: 可以使用瀏覽器的「另存為圖片」功能,或使用螢幕截圖工具。

### Q: 條碼/QR Code 為什麼是簡化的?

A: 這是前端模擬版本,提供視覺預覽。實際條碼內容已保留,可用於真實列印機。

### Q: 可以離線使用嗎?

A: 建置後即可離線使用,或設定為 PWA。
